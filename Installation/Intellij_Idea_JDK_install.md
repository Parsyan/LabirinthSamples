##Intellij Idea ներբեռնում և կարգավորում

Intellij Idea-ն կոդ գրելու միջավայր է ինչպես օնլայն ծրագիր աշխատացնող ծրագրերը, բայց այն իր մեջ ներառում է լրացուցիչ գործիքներ և հարմարանքներ որոնք հեշտացնում են ծրագրի ստեղծման գործընթացը։ Բայց այս գործիքի լավ աշխատանքի համար ձեր համակարգչում պետք է 8gb Ram(օպերատիվ հիշողություն) ունենաք և 6gb ազատ հիշողություն։ Այն կաշխատի 4gb Ram-ով ևս, բայց այն դանդաղ է աշխատելու։
Այն անվճար է, բայց ունի վճարովի տարբերակ ևս։

Ներբեռնելու համար ՝

`Մուտք գործեք Intellij Idea - ի պաշտոնական կայք փնտրելով համացանցում “Intellij Idea” - սա անվճար տարբերակն է։`

`Գտնված կայքերից առաջինները լինելու են JetBrains ընկերության կայքերը։ Այդ ընկերությունը Intellij Idea գործիքի ստեղծողն է։ Մուտք եք գործում “Download Intellij Idea” հղումով։`

`Կամ`

`Մուտք գործել Qr կոդով։`

Այս Էջում մի տարբերակ է լինելու։ Այն և վճարովի տարբերակն է և անվճար տարբերակը։ Դուք նրան կարող եք անվճար օգտագործել, եթե պետք լինի գնեք ձեզ համար վճարովի տարբերակի հնարավորությունները։ Ներբեռնեք՝ “Download”:


`Ներբեռնած ֆայլը փնտրեք ձեր համակարգչի “Downloads” կամ  “Загрузки” թղթապանակում 📁 “File Manager” կամ “Файловый Менеджер” ծրագրով։`
`Աշխատացրեք այդ ֆայլը մկնիկի երկու ձախ կլիկով։`
`Բացված պատուհանում ընտրեք “Next” երկու անգամ։`
`Ընտրեք “Intellij Idea” - ը, “.java”, “.gradle”, “.pom” վանդակները և ընտրեք “Next” մինչև չսկսի ծրագրի տեղադրումը։ Այն կարող է մի քանի րոպե տևել այնպես որ կարող եք գնալ սուրճի կամ թեյի ընդմիջման մինչև ավարտը կամ էլ ուղղակի սպասել։ `
`Ծրագրի տեղադրման ավարտից հետո կարող եք սեղմել “Run Intellij IDEA” վանդակին որպեսզի պատուհանը փակելուց հետո համակարգիչը աշխատացնի “Intellij IDEA” ծրագիրը`

Յուրաքանչյուր ծրագրավորման լեզու ունի իր ծրագիրը աշխատացնող համակարգը և այս ծրագիրը ներբեռնելուց հետո դուք պետք է նշեք կամ ներբեռնեք Java - ի՝ որ աշխատացնող ծրագիրն եք օգտագործելու։ Java - ի այդ ծրագրի անունը JDK (Java Development Kit) է որը նույնպես տրամադրում է Java - ի հետ աշխատելու համար գրադարաններ։

Եթե աշխատացնելու համար համակարգիչը թույլտվություն ցանկանա կարող եք տրամադրել այն անվտանգ ծրագիր է։

`Աշխատացնելուց հետո ծրագրի մեջ սեղմեք “New Project” աջի վերևի անկյունում։ Այն նոր ծրագիր ստեղծելու համար է։`

Բացված պատուհանի մեջ “Name:” - ի կողքին ձեր ծրագրի անունն է։ 

`Դուք կարող եք փոփոխել ինչպես կցանկանաք բայց անունը չպետք է թվից կամ հատուկ սիմվոլից սկսի և պետք է լինի լատինատառ։`

`Պատուհանի մեջ “JDK:” վանդակին սեղմելուց հետո փնտրեք և սեղմեք “Download JDK…” կոճակին։ Այն կընտրի Java - ի տիրոջ ստեղծած աշխատացնող ծրագրի վերջին տարբերակը։`
`  “Select” կոճակին սեղմեք և սպասեք մինչև ներբեռնման ավարտը կամ էլ եթե չի սկսել ներբեռնում սեղմեք “Create” կոճակը պատուհանի աջի ներքևի հատվածում։ `

Այն կստեղծի ձեր նոր ծրագիրը, կավելացնի կոդ փորձարկման համար և կսկսի ներբեռնել և տեղադրել ձեր ծրագրի մեջ JDK - ն։

Տեղադրումի ավարտից հետո կարող եք սեղմել ՝
 
`աշխատացնելու կոճակը և “Intellij Idea” - ն կաշխատացնի ձեր առաջին ծրագիրը։`

**Առաջին աշխատացնելու ընթացքում ձեզանից կարող է թույլտվություն ցանկանա համակարգիչը աշխատեցնելու համար կարող եք տալ այն անվտանգ է։**
